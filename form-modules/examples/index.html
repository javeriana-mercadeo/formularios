Sistema de formularios

Sistema modular para crear formularios de eventos de la Universidad Javeriana

Diseñado para ser fácil de usar, personalizable y completamente funcional. Construye formularios poderosos con una arquitectura modular y moderna.

FormManager.js config.js
// Inicializa un formulario con configuración básica
const form = new FormManager('form-id', {
  debugMode: true,
  autoSave: true
});

// Inicializa el formulario
form.initialize();

Comenzando

Todo lo que necesitas para empezar a usar Sistema de formularios en tu proyecto

1. Instalación

Incluye los módulos JavaScript necesarios en tu proyecto

<script src="modules/FormManager.js"></script>

2. Configuración

Define tu configuración inicial con los campos requeridos

const form = new FormManager('form-id', config);

3. Inicialización

Ejecuta el formulario y comienza a capturar datos

form.initialize();

FormManager

              Core
            

            

              Clase principal para gestionar formularios de eventos. Controla la inicialización,
              validación y envío de datos.
            


            
            

              
Constructor

              

                

                  
Método

                  
Parámetros

                  
Descripción

                

                

                  

                    new FormManager()
                  

                  

                    formId: string

                    config: object
                  

                  

                    Crea una nueva instancia del gestor de formularios
                  

                

              

            


            
            

              
Métodos

              

                

                  
Método

                  
Retorna

                  
Descripción

                

                

                  

                    initialize()
                  

                  

                    Promise<void>
                  

                  

                    Inicializa el formulario cargando datos, configurando validaciones y eventos
                  

                

                

                  

                    reset()
                  

                  

                    void
                  

                  

                    Restablece el formulario a su estado inicial, limpiando campos y errores
                  

                

                

                  

                    validate()
                  

                  

                    boolean
                  

                  

                    Valida todos los campos del formulario y muestra errores si los hay
                  

                

                

                  

                    getFormData()
                  

                  

                    object
                  

                  

                    Obtiene los datos actuales del formulario como un objeto
                  

                

                

                  

                    setDebugMode(enabled)
                  

                  

                    void
                  

                  

                    Activa o desactiva el modo debug para mostrar información detallada
                  

                

              

            

          


          
          

            

              
DataManager

              Data
            

            

              Módulo para el manejo de datos externos como países, departamentos, ciudades y
              programas académicos.
            


            

              
Métodos

              

                

                  
Método

                  
Retorna

                  
Descripción

                

                

                  

                    loadData(type)
                  

                  

                    Promise<Array>
                  

                  

                    Carga datos desde archivos JSON para poblar selectores
                  

                

                

                  

                    saveData(key, data)
                  

                  

                    void
                  

                  

                    Guarda datos en localStorage para persistencia local
                  

                

                

                  

                    clearData(key)
                  

                  

                    void
                  

                  

                    Elimina datos específicos del localStorage
                  

                

              

            

          


          
          

            

              
ValidationModule

              Validation
            

            

              Sistema de validación para campos de formulario con reglas personalizables y mensajes
              de error.
            


            

              
Métodos

              

                

                  
Método

                  
Retorna

                  
Descripción

                

                

                  

                    validateField(fieldName, value, rules)
                  

                  

                    ValidationResult
                  

                  

                    Valida un campo específico según las reglas proporcionadas
                  

                

                

                  

                    validateForm(formData)
                  

                  

                    ValidationSummary
                  

                  

                    Valida todos los campos del formulario y retorna un resumen de errores
                  

                

                

                  

                    showError(fieldName, message)
                  

                  

                    void
                  

                  

                    Muestra un mensaje de error visualmente en el campo especificado
